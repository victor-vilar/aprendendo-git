git restore <arquivo> - serve para descartar as alterações em um determinado arquivo.

git restore --staged <file> para desmarcar o arquivo para nao ser mais comitado.

git revert <rash> para retornar a um commit do rash passado.


git stash - pra salvar o arquivos temporariamente sem precisar dar um commit

git stash lista - mostra tudo o que esta na lista de stash

git stash apply<numero stash> - vai aplicar  as modificações so que elas vao ficar la na lista

git stash drop<numero> pra remover uma stash da lista.
